## Specs and Libraries used
- Minimum SDK 19
- Kotlin 
- MVVM Architecture
- DataBinding
- LiveData
- ViewModel
- Github Api
- RecyclerView
- [Retrofit2 & Moshi](https://github.com/square/retrofit) for constructing the REST API
- [Glide](https://github.com/bumptech/glide) for loading images
